## Initial commit — Project Foundation
**Commit:** `c3896ae9` · **Date:** 2026-03-04 · **Author:** Vanessa Aristizabal

This is the primary documentation entry for the **AI Playbook** repository.

### What was created

A reusable, client-agnostic AI Playbook repository designed to improve developer performance when working with AI coding assistants (GitHub Copilot, Claude, Cursor). The playbook provides:

- **Agents** — multi-step procedures that orchestrate skills and tools
- **Skills** — procedural files following the [agentskills.io](https://agentskills.io) structure
- **Policies** — rule files applied by agents to enforce project standards
- **Tools** — shared Node.js CLI utilities (project detection, scope guard, diagnostics)
- **React + TypeScript application** — visual guide and interactive demo for the playbook

#### Repository structure

```
ai-playbook-reposito/
├── .github/
│   ├── copilot-instructions.md   # GitHub Copilot routing instructions
│   └── dependabot.yml            # Automated dependency updates
├── ai-playbook/
│   └── .github/
│       ├── agents/               # Agent definitions (AGENT.md files)
│       ├── copilot-instructions/ # Policy files (frontend, backend, workspace, style)
│       ├── skills/               # Skill definitions (SKILL.md + references + assets + scripts)
│       └── specs/                # Specification templates
├── tools/                        # Shared Node.js utilities
├── docs/                         # Project documentation
├── src/                          # React + TypeScript application source
├── CLAUDE.md                     # Claude AI instructions
├── .cursorrules                  # Cursor IDE instructions
├── LICENSE                       # MIT License
├── README.md                     # Full project documentation
└── SECURITY.md                   # Security policy
```

#### Tech stack

- **React 19 + TypeScript 5** with Vite
- **Tailwind CSS 4** for styling
- **shadcn/ui** as the component foundation
- **Vitest + React Testing Library + jest-axe** for testing
